# www-venetian-guide

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式保存，不依赖于任何特定后端服务，也不针对某个单一域名或网站。

## 项目简介

本项目是一个静态 HTML 页面归档集合。你可以将其视为一个轻量级的页面容器，用于存放、组织和展示一系列独立的 HTML 文件。每个页面都是自包含的，适合长期保存或离线浏览。

## 目录结构

```
.
├── index.html          # 仓库首页（可选）
├── pages/              # 归档页面存放目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
└── README.md           # 本文件
```

- `index.html`：仓库入口页面，可根据需要创建，用于导航或展示。
- `pages/`：主要页面归档目录，所有独立 HTML 文件均放置于此。

## 页面归档说明

- 每个 HTML 文件均为独立完整的页面，包含必要的样式和脚本。
- 归档页面不依赖外部资源（或已内联），以便长期保存。
- 文件名建议使用有意义的英文或拼音，避免特殊字符。
- 如需添加新页面，直接放入 `pages/` 目录并更新索引即可。

## 使用方式

你可以通过以下方式访问和浏览归档页面：

1. **直接打开本地文件**：克隆仓库后，在浏览器中打开 `pages/` 下的任意 HTML 文件。
2. **通过 GitHub Pages 部署**：若启用 GitHub Pages，仓库根目录或 `docs/` 目录下的内容将自动发布为静态站点。
3. **自行托管**：将仓库内容上传至任意静态文件服务器。
